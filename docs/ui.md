# UI (User Interface)

UI — это слой взаимодействия пользователя с системой Home-SCADA.

UI не содержит бизнес-логики и не принимает решений.
Все решения принимаются в ядре системы.

---

## UI v0 (минимальный)

### UI отображает:
- текущее значение сенсора
- состояние сенсора (OK / WARN / ALARM)
- состояние актуатора (ON / OFF)

### UI позволяет:
- включить актуатор
- выключить актуатор

---

## Принцип работы

UI читает состояние ядра
UI отправляет команды в ядро
